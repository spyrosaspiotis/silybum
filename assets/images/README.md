# Image assets

Use only the final approved Silybum visuals.

Required filenames:

- `hero.webp` — main hero image
- `easy.webp` — Easy Collection card
- `comfort.webp` — Comfort Collection card
- `luxury.webp` — Luxury Collection card

Recommended export:

- WebP format
- sRGB colour profile
- hero: at least 2200 px wide
- collection images: at least 1200 px wide
- quality: 82–88%

Do not include chandeliers or white mannequins. Collection photography should preserve the approved floral style, floral arch placement at the building entrance, ivory wedding runner, candles and the distinct quantity/scale of arrangements for each collection.
